gumby
=====

[![version](https://img.shields.io/pypi/v/gumby.svg?style=flat-square)][pypi]

[pypi]: https://pypi.org/project/gumby/

Stretch polygonal meshes in segments along an axis.

Installation
------------

Requires Python 2.7.

```sh
pip install gumby
```
